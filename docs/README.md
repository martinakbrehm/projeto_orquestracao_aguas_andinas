# Conhecimento Replicável — projeto_banco_neo

Esta pasta centraliza documentos de conhecimento capturado durante o desenvolvimento do projeto. O objetivo é permitir replicação, adaptação e validação automatizada (inclusive por LLMs) sem depender de memória pessoal ou histórico de conversa.

## Arquivos

| Arquivo               | Conteúdo                                                                 |
|-----------------------|--------------------------------------------------------------------------|
| [dashboard_macro.md](dashboard_macro.md) | Decisões técnicas, de design e de dados do dashboard de aproveitamento. Inclui schema, queries, arquitetura, migrations e checklist de replicação. |

## Como usar

- Para replicar o dashboard em outro contexto: siga o checklist da seção 8 de `dashboard_macro.md`
- Para debugar problemas de dados: consulte a seção 9 (problemas já resolvidos)
- Para alimentar uma LLM: forneça o arquivo completo como contexto e faça perguntas específicas sobre a seção relevante
